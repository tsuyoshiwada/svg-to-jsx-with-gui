## Development

Install depedencies libraries.

```bash
$ npm install
```

Run the tool locally. You can access it with http://localhost:3000 .

```bash
$ npm start
```

Unit testing, and Lint and Typecheck (flow).

```bash
$ npm test
```

Build the tool.

```bash
$ npm run dev
```

Production build.

```bash
$ npm run prod
```

Deploy to GitHub Pages.

```bash
$ npm run deploy
```
