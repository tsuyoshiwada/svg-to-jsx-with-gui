## Development

Install depedencies libraries.

```bash
$ npm install
```

Run the tool locally. You can access it with http://localhost:3000 .

```bash
$ npm start
```

Unit testing, and Lint.

```bash
$ npm test
```

Build the tool.

```bash
$ npm run build
```

Deploy to GitHub Pages.

```bash
$ npm run deploy
```
